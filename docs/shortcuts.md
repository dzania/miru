# Keyboard Shortcuts

Complete reference for every key binding in miru.

## Reader

| Key | Action |
|-----|--------|
| `j` | Scroll down one line |
| `k` | Scroll up one line |
| `d` | Scroll down half a page |
| `u` | Scroll up half a page |
| `g` | Jump to top of document |
| `G` | Jump to bottom of document |
| `/` | Enter search mode |
| `n` | Jump to next search match |
| `N` | Jump to previous search match |
| `h` | Switch to previous tab |
| `l` | Switch to next tab |
| `Tab` | Open headings index |
| `Ctrl+T` | Open file picker (new tab) |
| `w` | Close current tab |
| `?` | Toggle the help bar |
| `Backspace` | Return to file picker |
| `q` | Quit miru |

## Search Bar

| Key | Action |
|-----|--------|
| *any char* | Append to search query |
| `Backspace` | Delete last character |
| `Enter` | Confirm query and close bar |
| `Escape` | Clear query and close bar |

## File Picker

| Key | Action |
|-----|--------|
| `j` / `k` | Navigate list |
| `/` | Enter filter mode |
| `q` | Quit miru |

### Filter mode

| Key | Action |
|-----|--------|
| *any char* | Append to filter query |
| `Backspace` | Delete last character |
| `Enter` | Confirm filter — switch to navigate |
| `Escape` | Cancel filter — discard query |

### Navigate mode (after confirming filter)

| Key | Action |
|-----|--------|
| `j` / `k` | Move through filtered results |
| `Enter` | Open selected file |
| `Escape` | Clear filter — show all files |
| `/` | Enter filter mode again |

## Headings Index

| Key | Action |
|-----|--------|
| `j` | Move selection down |
| `k` | Move selection up |
| `g` | Jump to first heading |
| `G` | Jump to last heading |
| `Enter` | Jump to selected heading |
| `Tab` | Close index |
| `Escape` | Close index |
| `q` | Quit miru |
