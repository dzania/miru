# miru

Minimal markdown reader I use daily.


## Usage

```
miru [files...]
```

Open files directly or run `miru` in a directory to pick from `.md` files.

## Keys

### Reader

| Key | Action |
|-----|--------|
| `j/k` | Scroll |
| `d/u` | Page down/up |
| `g/G` | Top/bottom |
| `/` | Search |
| `n/N` | Next/prev match |
| `h/l` | Prev/next tab |
| `Tab` | Headings index |
| `Ctrl+T` | Open file |
| `w` | Close tab |
| `?` | Toggle help |
| `q` | Quit |

### Picker

| Key | Action |
|-----|--------|
| `j/k` | Navigate |
| `/` | Filter |
| `Enter` | Open |
| `q` | Quit |

## Install

```
cargo install --path .
```
